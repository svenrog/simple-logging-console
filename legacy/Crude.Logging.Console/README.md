# Crude.Logging.Console (deprecated)

This package has been renamed to [`Simple.Logging.Console`](https://www.nuget.org/packages/Simple.Logging.Console) and will receive no further updates.

Update your reference:

```
dotnet remove package Crude.Logging.Console
dotnet add package Simple.Logging.Console
```

The namespace also changed from `Crude.Logging.Console` to `Simple.Logging.Console`, and `AddConsoleLogging()` moved with it — no other code changes are required.

See the [project repository](https://github.com/svenrog/simple-logging-console) for details.
